javascript 문서 사용하는 위치 파악하기

javascript는 제어의 기능을 하는 언어 : html/css문서의 내용을 상ㅂ입/삭제/수정
서버를 구성하여 처리하기, 동기화처리, animation/버튼클릭시 처리되는 기능

javascript는 java언어와는 전혀 다른 언어로 기능부터 모든것이 다르다

다양한 버전과 다양한 스크립트 언어들이 존제

종류는 : javascript, coffeescript, actionscript, typescript, dart, ecmascript ...

버전 : ecma의 표준 기반으로 다양한 버전이 존재 es5/es6 이상
